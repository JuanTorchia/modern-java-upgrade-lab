# Java 8 To 17 Rules Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the first real migration rule architecture and Java 8 to Java 17 findings.

**Architecture:** `analyzer-core` gets a lightweight rule engine and default analyzer. `cli` keeps Maven inspection and rendering, but delegates finding creation to `DefaultAnalyzer`. The report format remains stable.

**Tech Stack:** Java 25, Maven, JUnit 5, AssertJ, Picocli.
