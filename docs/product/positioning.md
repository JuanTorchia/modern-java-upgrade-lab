# Product Positioning

Modern Java Upgrade Lab is an open source migration-readiness CLI for Java teams that need evidence before changing runtime, framework, build, or CI baselines.

## Primary User

The first target user is a Tech Lead or Staff Engineer responsible for making a Java upgrade credible to engineering leadership, security, platform, and delivery teams.

## Open Source Core

- Local CLI analysis for Maven and Gradle projects.
- Markdown and JSON migration-readiness reports.
- Java 8/11/17/21/25 LTS scenario coverage.
- Build, dependency, runtime, source-pattern, and CI readiness signals.
- Portfolio summaries generated from local JSON reports.
- OpenRewrite recipe recommendations where automation is useful.

## Why Teams Would Care

Teams do not pay for another generic scanner. They pay to reduce migration uncertainty:

- Which repos are blocked?
- Which Java/Spring/build/runtime baselines are risky?
- Which work items should be done first?
- Which changes are safe to automate?
- Which optional refactors should stay out of the migration branch?

## Future Paid-Worthy Extensions

- Hosted dashboard for GitHub/GitLab organizations.
- Multi-repo scanning with historical trend tracking.
- Pull request orchestration for OpenRewrite and custom recipes.
- Enterprise rule packs for internal frameworks and platform standards.
- Signed migration assessment reports for consulting/support engagements.

The open source project should remain useful without any hosted service. Commercial value should come from scale, governance, custom policy, reporting, and support.
